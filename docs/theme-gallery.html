Theme gallery: eight directions, same app

Every mockup renders the identical Intent Studio slice, so you are comparing pure identity: palette, type voice, radii, and the login backdrop treatment. Pick one light and one dark (the app ships with a theme toggle), or a single direction if you'd rather ship one theme. Fonts shown are system stand-ins; the real faces are named per direction and will be self-hosted.

Light rooms

Daylight desks, dense reading. Four different temperaments.

A

Proof Room

Schibsted Grotesk + Spline Sans Mono

Your earlier pick, revised: paper-white, warm graphite ink (meta text darkened as you flagged), one deep carmine stamp. The quiet editorial cousin of the bank's red.

C

Gallery

Neue Haas / Helvetica Now + IBM Plex Mono

Swiss white-cube minimalism: pure white, near-black ink, a single cobalt klein-blue accent, near-square corners. The most severe and most timeless; typography does all the work.

E

Celadon Counter

Onest + Commit Mono

Celadon-glazed surfaces, deep spruce ink and accent, a whisper of persimmon in the login scene. Botanical calm for long sessions; the least "banking" of the set, in a good way.

F

Print Shop

Suisse-style grotesk + SF Mono, uppercase page titles

Industrial: cool gray paper, jet-black controls, chartreuse highlighter for selection and the run track. Squared corners, heavier rules. The boldest light option; reads like a working pressroom.

Dark rooms

After-hours instruments. Four different signals, no orange anywhere.

B

Night Desk

Schibsted Grotesk + Spline Sans Mono

Your earlier pick, revised: warm graphite-black with the orange swapped for a luminous carmine-rose, the same bloodline as Proof Room's stamp. Pairs seamlessly with A.

D

Midnight Archive

Source Serif 4 titles + IBM Plex Sans/Mono

A private bank's vault after hours: deep indigo surfaces, brass-gold signal, serif page titles. Heritage and gravitas; approvals feel like signing a ledger.

G

Deep Harbor

Hanken Grotesk + JetBrains Mono

Blue-black harbor night with a cyan beacon. Cool, technical, wide-awake; the closest to a modern observability tool while staying distinctly quieter.

H

Ultraviolet Console

Geist-family grotesk + Fira Code

Graphite dusk with an iris-violet signal and a faint magenta afterglow in the login scene. Contemporary product-tool energy; the friendliest dark option.
