Style round: three structures, one skin

Theme is now fixed (Gallery light + Deep Harbor dark). This round is design style: how the same Intent Studio content is structured. All three shown in light; your chosen dark theme applies to whichever you pick. The style carries across all six pages.

1 · Instrument = everything on one screen, toolbar-driven, maximum density. 2 · Studio = editorial two-column with a dedicated generation rail, most whitespace. 3 · Console = split list/detail workbench with a command dock, most "power tool".

STYLE 1

Instrument

Single scrolling column: toolbar over a dense ruled table, run card directly beneath. Least navigation, most information per screen — the shape of Stripe's dashboard tables. Generation opens as an inline panel above the table when triggered.
