Intent Generation Backoffice — four visual directions

Each direction renders the exact same slice of the app: the shell with global project switcher, the Intent Studio workspace (topic scope, source table, batch run in flight), and the login backdrop treatment. Content is identical everywhere — you're choosing an identity, not a layout.

Previews use system font stand-ins; the real typefaces named in each spec strip will be self-hosted in the build. Every direction keeps the login orb-backdrop you like, recolored to its own world.

DIRECTION A

Proof Room

A typesetter's proof room in a private bank: paper-white ground, warm graphite ink, one deep carmine stamp. The quiet cousin of the bank's red — calm, precise, editorial in the details. Closest in spirit to Stripe's dashboard with a stronger typographic voice.

Type: Schibsted Grotesk (UI) + Spline Sans Mono (data) Feel: instrument, warm-neutral, softly squared (8px)
DIRECTION B

Night Desk

A dark control room for knowledge ops: graphite-black surfaces, ember-orange signal, mono-heavy metadata. Feels like a refined trading terminal — Linear's dark mode crossed with an instrument panel. The generation run glows; everything else recedes.

Type: Hanken Grotesk (UI) + JetBrains Mono (data) Feel: terminal-native, high signal, tight radii (6px)
DIRECTION C

Ledger

A banking ledger set by an editorial studio: ink-navy text on cool porcelain, oxblood accent, serif page titles over ruled tables, near-square corners. The most "institutional heritage" of the four — governance rendered as fine print done beautifully.

Type: Source Serif 4 (titles) + IBM Plex Sans (UI) + IBM Plex Mono (data) Feel: engraved, ruled, crisp (4px)
DIRECTION D

Verdigris

A clean break from red altogether: deep verdigris green as the working color on cool white, generous 12px radii, the friendliest and most contemporary-SaaS of the four. Calming for long sessions; approvals and "indexed" states feel native to the palette.

Type: Onest (UI) + Fira Code (data) Feel: soft, modern, approachable (12px)
